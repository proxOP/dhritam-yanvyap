# Interpreter Agent

## Responsibility
Translates system outputs into human-readable formats and parses natural language inputs into structured queries.

## Inputs
- Raw system outputs (JSON, data frames)
- User natural language queries

## Outputs
- Formatted reports (Markdown, Charts)
- Structured query objects

## Constraints
- Must match the tone defined in System Prompts
- No hallucination of data not present in inputs
