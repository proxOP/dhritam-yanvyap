# Dritam: AI Financial Intelligence System

**Phase 1: Scaffolding**

This repository contains the architectural skeleton for `dritam`, an AI-driven system for financial market and supply chain analysis.

## Structure

- `api/`: REST interface and request lifecycle definitions.
- `agents/`: Specifications for all autonomous agents (Observer, Market, SupplyChain, etc.).
- `graph/`: Definition of the Reasoning Graph logic.
- `memory/`: Schemas for episodic and semantic memory storage.
- `prompts/`: System-level prompts and personality definitions.
- `policies/`: Ethics and safety guardrails.
